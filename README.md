
## facebook 2023 login page clone

facebook 2023 login page clone using tailwindcss


### Installation

Run using vite 

```bash
  git clone https://github.com/thwahatkp/fb-clone.git
  cd fb-clone
  npm install
  npm start
```
or open dist folder and run using liveserver
